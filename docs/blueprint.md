# **App Name**: A2G Smart Notes

## Core Features:

- Notes Library: Provides access to a structured notes library, organized by course, year and subject.
- AI Notes Generator: Allows premium users to generate notes on-demand using AI. After generation, the AI tool can answer follow-up questions using conversational AI. It is a tool.
- AI Exam Question Generator: Generates potential exam questions and provides AI insights on their likelihood, frequency and previous years asked. It is a tool.
- Academic Services Hub: Showcases all academic writing and project support services offered via service cards in a grid.
- Store: Facilitates navigation to affiliate products and internal PDF bundles through a tabbed interface (Books, Courses, Bundles).
- Premium Plans: A page showing subscription plans, emphasizing the benefits of each, to convert free users to premium subscribers.
- Notifications: Provides notifications for university and exam-related announcements.

## Style Guidelines:

- Primary color: A vibrant violet (#9400D3) to convey intelligence and creativity, resonating with pharmacy students.
- Background color: Light lavender (#E6E6FA), providing a soft and clean backdrop to the violet primary.
- Accent color: Deep purple (#4B0082) to highlight key interactive elements, such as calls-to-action and buttons, ensuring clarity and focus.
- Body text font: 'PT Sans', a modern humanist sans-serif, is suitable for body text.
- Headline font: 'Playfair', a modern serif font with high contrast, is suitable for headlines.
- Use clean, simple icons to represent different categories and subjects in the notes library and other sections, aiding navigation and visual appeal.
- Implement subtle transitions and animations on button clicks and page loads to enhance the user experience without being distracting.
