# **App Name**: phamA2G

## Core Features:

- Dashboard: Clean and intuitive dashboard providing access to all features.
- Note Management: Display and manage notes with options for premium content indicated.
- Responsive Design: Responsive layout adapting to different screen sizes.
- AI Note Generation: AI tool to generate notes based on course, year, subject, and topic inputs, offering structured Markdown content.
- AI Quiz Generation: Tool that automatically generates quizzes based on existing notes, assessing a user's understanding of specific subjects and topics.
- User Authentication: Allow users to log in/sign up with email/password or using Google OAuth.
- Newsletter Subscription: Manage newsletter subscriptions using Firebase.

## Style Guidelines:

- Primary color: Deep blue (#3B5998) to reflect trustworthiness.
- Background color: Light blue (#E6EBF5) to give a sense of calm and spaciousness.
- Accent color: Orange (#FFA500) to highlight interactive elements and important actions.
- Body font: 'Inter' (sans-serif) for clean and modern readability.
- Headline font: 'Space Grotesk' (sans-serif) for impactful titles.
- Use vector-based icons from Lucide to ensure clarity and scalability.
- Implement subtle animations to enhance user interaction, such as loading indicators and button hover effects.
